# Inception (Начало)
Игра в жанре песочницы о добыче ресурсов и расширении своего острова. Сделано для курсовой работы по дисциплине МДК 01.01 Разработка программных модулей. 

## Особенности:

### Выращивание растений:
Возможность выращивать различные растения, которые используются для создания еды и предметов.

### Создание предметов:
Ввозможность создавать различные типы предметов, которые помогают в процессе увеличения добычи ресурсов.

### Выполнение заданий:
На протяжении всего игрового процесса игрок будет получать задания, которые ему требуется выполнить, чтобы сохранить свою территорию.

### Выживание:
Для поддержания жизни игроку постоянно будет требоваться добывать еду и противостоять существам, что населяют этот мир.
